# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-20

### Added
- **Interactive Editor**: Built comprehensive forms for Personal, Experience, Education, Skills, and Projects.
- **Live Preview**: Implemented real-time resume preview.
- **PDF Export**: Added functionality to download resume as PDF.
- **Branding**: Added new "Resumify" logo and site identity.
- **Documentation**: Added README, LICENSE, and CONTRIBUTING guide.
- **Deployment**: Configured and deployed to GitHub Pages.

### Fixed
- **Routing**: Resolved 404 and routing errors on GitHub Pages by configuring `basename`.
- **Layout**: Fixed syntax errors and component structure in EditorLayout.

### Technical
- Migrated from CRA scripts to Vite for better performance.
- Configured Tailwind CSS for styling.
